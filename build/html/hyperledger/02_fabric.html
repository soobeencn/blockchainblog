2. Fabric 简介¶

2.1. Hyperledger简介¶

  • 通用行业、通用业务场景的企业级区块链解决方案
  • 多项目并行

2.2. Fabric简介¶

  • IBM发起、Linux基金会托管
  • 企业级联盟链基础设施
  • 可插拔的共识机制(solo、kafka…)
  • 多链多通道隔离

2.3. 智能合约¶

  • 区块链2.0 : 以太坊
  • 合约协议的数字化代码表达
  • 分布式有限状态机
  • 执行环境安全隔离、不受第三方干扰(EVM、Docker)

2.4. 链码¶

  • Fabric应用层基石(中间件)
  • 链码编程接口
    • init() 只执行一次
    • invoke() 应用程序与链码交互的入口
    • 上述两个方法都只接受同一个参数,SDK链码交互的接口
